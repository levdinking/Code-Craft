# Agent Instructions

## Language
Always respond in Russian. All explanations, comments, and communications must be in Russian.
Code comments should also be in Russian where appropriate.

## Project
- Stack: React 19 + TypeScript + Vite + Tailwind CSS + Framer Motion
- Admin server: `npm run admin-server` (Express on port 3001)
- Dev: `npm run dev`
- Build: `npm run build`
- TypeScript check: `npx tsc -b --noEmit`
